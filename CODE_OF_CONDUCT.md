# Code of Conduct

This project follows the spirit of the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version:

- **Be kind and patient.** Many users and contributors here are beginners with
  git — that's the whole point of the project. There are no stupid questions.
- **Be respectful.** Disagree with code, not with people. No harassment,
  discrimination, personal attacks, or unwelcome sexual attention — anywhere
  the project happens (issues, PRs, discussions, previews).
- **Assume good intent, and own your mistakes.** Apologise, fix it, move on.

Maintainers may edit or remove comments, commits, issues and PRs that violate
this, and may temporarily or permanently ban contributors for behavior they
deem inappropriate, threatening, or harmful.

**Reporting:** email the maintainer (see the GitHub profile of
[@Abhialphavima007](https://github.com/Abhialphavima007)) or use GitHub's
*Report content* tools. All reports are handled confidentially.
